LCG(linear congruential generator,线性同余算法)

介绍

一种生成为随机数的算法

公式是

Rn+1=(aRn+b) mod m,其中a,b,m是随机常量,R0称为随机数种子R_{n+1} = (a R_n + b) \ mod \ m,\\ 其中a,b,m是随机常量,R_0称为随机数种子 Rn+1​=(aRn​+b) mod m,其中a,b,m是随机常量,R0​称为随机数种子

其周期最大为M，当且仅当：

1. gcd(B,M)=12. ∀素数p，若p∣m，则p∣a−13. 若4∣m，则4∣a−14. a,b,R0<m\begin{aligned} 1.& \ gcd(B,M) = 1 \\ 2.& \ \forall 素数p，若p|m，则p|a-1\\ 3.& \ 若4|m，则4|a-1 \\ 4.& \ a,b,R_0 < m \end{aligned} 1.2.3.4.​ gcd(B,M)=1 ∀素数p，若p∣m，则p∣a−1 若4∣m，则4∣a−1 a,b,R0​<m​

简陋的代码

应用于Crypto

加密的例子：

$ R_{n + 1} 反推 R_n$​

Rn+1≡aRn+b (mod m)故Rn≡a−1Rn+1−b (mod m)R_{n+1} \equiv a R_n + b \ ( mod \ m )\\ 故 R_n \equiv a^{-1} R_{n + 1} - b \ ( mod \ m ) Rn+1​≡aRn​+b (mod m)故Rn​≡a−1Rn+1​−b (mod m)

$ 求a $​

Rn+1≡aRn+b (mod m)Rn+2≡aRn+1+b (mod m)Rn+2−Rn+1≡a(Rn+1−Rn) (mod m)故a=(Rn+2−Rn+1)(Rn+1−Rn)−1 (mod m)R_{n + 1} \equiv a R_n + b \ ( mod \ m )\\ R_{n + 2} \equiv a R_{n + 1} + b \ ( mod \ m )\\ R_{n + 2} - R_{n + 1} \equiv a (R_{n + 1} - R_{n}) \ ( mod \ m )\\ 故 a = (R_{n + 2} - R_{n + 1}) (R_{n + 1} - R_{n})^{-1} \ ( mod \ m )\\ Rn+1​≡aRn​+b (mod m)Rn+2​≡aRn+1​+b (mod m)Rn+2​−Rn+1​≡a(Rn+1​−Rn​) (mod m)故a=(Rn+2​−Rn+1​)(Rn+1​−Rn​)−1 (mod m)

$ 求m $​

记tn=Rn+1−Rn有atn≡tn+1 (mod m)tn2≡tn+1tn+2 (mod m)令Tn=tn+1tn+2−tn2，有Tn≡0 (mod m)故m=gcd(Tn+1,Tn)记 t_n = R_{n + 1} - R_n\\ 有 a t_n \equiv t_{n+1} \ ( mod \ m )\\ t_n^2 \equiv t_{n+1} t_{n+2} \ ( mod \ m )\\ 令 T_n = t_{n+1} t_{n+2} - t_n^2 ，\\ 有 T_n \equiv 0 \ ( mod \ m )\\ 故 m = gcd (T_{n+1}, T_n) 记tn​=Rn+1​−Rn​有atn​≡tn+1​ (mod m)tn2​≡tn+1​tn+2​ (mod m)令Tn​=tn+1​tn+2​−tn2​，有Tn​≡0 (mod m)故m=gcd(Tn+1​,Tn​)
