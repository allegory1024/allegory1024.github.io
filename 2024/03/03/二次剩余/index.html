原根

阶的定义

对于整数 $$a,p,(a,p)=1$$
若 $$x$$ 为使

ax≡1(mod p)a^x \equiv 1(mod \ p) ax≡1(mod p)

成立的最小正整数
则称 $$x$$ 为 $$a$$ 模 $$p$$ 的阶，记为 $$e_p(a)$$
(有些书上记为 $$ord_pa$$ )

原根的定义

若$$e_p(a)=p-1,$$ 则称 $$a$$ 为模 $$p$$ 的原根

原根定理

\forall$$ 素数 $$p$$ , 恰好有 $$\phi(p-1)$$ 个模 $$p$$ 的原根 **证明：** 记 $$\psi(d)=\lvert\{a|1\leq a<p , e_p(a)=d \}\rvert,

特别地，$$\psi(p-1)$$ 为模 $$p$$ 的原根个数

引理1：

对 $$\forall n|p-1, $$

ψ(d1)+ψ(s2)+...+ψ(dk)=n\psi(d_1) + \psi(s_2) + ... + \psi(d_k) = n ψ(d1​)+ψ(s2​)+...+ψ(dk​)=n

引理1的证明：

设$$p=nk$$

Xp−1−1=Xnk−1=(Xn)k−1=(Xn−1)((Xn)k−1+(Xn)k−2+...+Xn+1)\begin{aligned} X^{p-1}-1=&X^{nk}-1\\ =&(X^n)^k-1\\ =&(X^n-1)((X^n)^{k-1}+(X^n)^{k-2}+...+X^n+1)\\ \end{aligned} Xp−1−1===​Xnk−1(Xn)k−1(Xn−1)((Xn)k−1+(Xn)k−2+...+Xn+1)​

其中$$1 \leq X \leq p-1\$$

一方面,

由费马小定理知，

Xp−1≡1(mod p)X^{p-1} \equiv 1 (mod \ p) Xp−1≡1(mod p)

恰好有 $$X=1,2,...,p-1$$ 这$$p-1$$个解，而

(Xn)k−1+(Xn)k−2+...+Xn+1≡0(mod p)(X^n)^{k-1}+(X^n)^{k-2}+...+X^n+1 \equiv 0 (mod \ p) (Xn)k−1+(Xn)k−2+...+Xn+1≡0(mod p)

至多$$nk-n$$个解

Xn−1≡0(mod p)X^n-1 \equiv 0(mod \ p) Xn−1≡0(mod p)

至多$$n$$个解

故显然

Xn≡1(mod p)X^n \equiv 1 (mod \ p) Xn≡1(mod p)

恰好$$ n $$个解

另一方面,
若 $$X=a$$ 是

Xn≡1(mod p)X^n \equiv 1(mod \ p) Xn≡1(mod p)

的解 , 则

ep(a)∣n,e_p(a)|n, ep​(a)∣n,

设 $$n $$ 的所有因子为 $$ d_1,d_2,...,d_k$$​ , 则

Xn≡1(mod p)X^n \equiv 1(mod \ p) Xn≡1(mod p)

的解的个数等于 $$ \psi(d_1) + \psi(d_2) + ... + \psi(d_k)$$

故

ψ(d1)+ψ(s2)+...+ψ(dk)=n \psi(d_1) + \psi(s_2) + ... + \psi(d_k) = n ψ(d1​)+ψ(s2​)+...+ψ(dk​)=n

引理2：

对于素数q∣p−1,有ψ(q)=ϕ(q)对于素数q|p-1,有\psi(q)=\phi(q) 对于素数q∣p−1,有ψ(q)=ϕ(q)

引理2的证明：

由引理1，

ϕ(q)+ϕ(1)=q=ψ(q)+ψ(1)\phi(q) + \phi(1) = q = \psi(q) + \psi(1) ϕ(q)+ϕ(1)=q=ψ(q)+ψ(1)

而

ϕ(1)=ψ(1)\phi(1) = \psi(1) ϕ(1)=ψ(1)

故

ϕ(q)=ψ(q)\phi(q) = \psi(q) ϕ(q)=ψ(q)

引理3：

对素数 $$q_1,q_2|p-1$$

ϕ(q1q2)=ψ(q1q2)\phi(q_1q_2) = \psi(q_1q_2) ϕ(q1​q2​)=ψ(q1​q2​)

引理3的证明：

由引理1，

ϕ(q1q2)+ϕ(q1)+ϕ(q2)+ϕ(1)=ψ(q1q2)+ψ(q1)+ψ(q2)+1 \phi(q_1q_2) + \phi(q_1) + \phi(q_2) + \phi(1) = \psi(q_1q_2) + \psi(q_1) + \psi(q_2) + 1\\ ϕ(q1​q2​)+ϕ(q1​)+ϕ(q2​)+ϕ(1)=ψ(q1​q2​)+ψ(q1​)+ψ(q2​)+1

由引理2，

ϕ(q1)=ψ(q1) , ϕ(q2)=ψ(q2)\phi(q_1) = \psi(q_1) \ , \ \phi(q_2) = \psi(q_2) ϕ(q1​)=ψ(q1​) , ϕ(q2​)=ψ(q2​)

故

ϕ(a1q2)=ψ(q1q2)\phi(a_1q_2) = \psi(q_1q_2) ϕ(a1​q2​)=ψ(q1​q2​)

回到原定理的证明，

由引理3，设n的所有因子为d1,d2,...,dkd_1,d_2,...,d_kd1​,d2​,...,dk​

ϕ(d1)+ϕ(d2)+...+ϕ(dk)=n=ψ(d1)+ψ(d2)+...+ψ(dk) \phi(d1) + \phi(d_2) + ... + \phi(d_k) = n = \psi(d_1) + \psi(d_2) + ... + \psi(d_k) ϕ(d1)+ϕ(d2​)+...+ϕ(dk​)=n=ψ(d1​)+ψ(d2​)+...+ψ(dk​)
