连分数（continued fractions ）

对于∀pq∈Q,∃n,a0,a1,a2,...,an,st.pq=a0+1a1+1a2+1...1an−1+1an=[a0,a1,a2,...,an]对于\forall \frac{p}{q} \in Q, \\ \exists n,a_0, a_1, a_2, ..., a_n,\\ st.\frac{p}{q} = a_0 + \frac{1}{a_1 + \frac{1}{a_2 + \frac{1}{... \frac{1}{a_{n-1} + \frac{1}{a_n}}}}}\\ = [a_0, a_1, a_2, ... , a_n] 对于∀qp​∈Q,∃n,a0​,a1​,a2​,...,an​,st.qp​=a0​+a1​+a2​+...an−1​+an​1​1​1​1​1​=[a0​,a1​,a2​,...,an​]

对有理数pq,取a0,r0∈N,st.p=a0⋅q+r0 (r0<q)故pq=a0+1qr0同理，取a1,r1∈N+,st.q=a1⋅r0+r1  (r1<r0)有qr0=a1+1r0r1，即pq=a0+1a1+r0r1如此，可取ai, i=0,1,2,...以下证明ai为有限项，（其实就是辗转相除法）对于ri=ai+2⋅ri+1+ri+2由ai+1≥1,知ri+2<ri+1≤ai+2⋅ri+1故ri+2<12ri因此，至多算2log⁡2q次(其实不用，但是我们只需要知道有限就够了)对有理数\frac{p}{q},\\ 取 a_0, r_0 \in \mathbb{N}, st. p = a_0 \cdot q + r_0 \ (r_0 < q)\\ 故\frac{p}{q} = a_0 + \frac{1}{\frac{q}{r_0}}\\ 同理，取 a_1, r_1 \in \mathbb{N^+}, st. q = a_1 \cdot r_0 + r_1 \ \ (r_1 < r_0)\\ 有\frac{q}{r_0} = a_1 + \frac{1}{\frac{r_0}{r_1}}，即\frac{p}{q} = a_0 + \frac{1}{a_1 + \frac{r_0}{r_1}}\\ 如此，可取a_i, \ i = 0,1,2,...\\ 以下证明a_i为有限项，（其实就是辗转相除法）\\ 对于r_i = a_{i + 2} \cdot r_{i + 1} + r_{i + 2}\\ 由a_{i + 1} \geq 1,知r_{i + 2} < r_{i + 1} \leq a_{i + 2} \cdot r_{i + 1}\\ 故r_{i + 2} < \frac{1}{2} r_i\\ 因此，至多算2\log_{2}{q}次(其实不用，但是我们只需要知道有限就够了) 对有理数qp​,取a0​,r0​∈N,st.p=a0​⋅q+r0​ (r0​<q)故qp​=a0​+r0​q​1​同理，取a1​,r1​∈N+,st.q=a1​⋅r0​+r1​  (r1​<r0​)有r0​q​=a1​+r1​r0​​1​，即qp​=a0​+a1​+r1​r0​​1​如此，可取ai​, i=0,1,2,...以下证明ai​为有限项，（其实就是辗转相除法）对于ri​=ai+2​⋅ri+1​+ri+2​由ai+1​≥1,知ri+2​<ri+1​≤ai+2​⋅ri+1​故ri+2​<21​ri​因此，至多算2log2​q次(其实不用，但是我们只需要知道有限就够了)

勒让德定理（Legendre’s theorem）

当满足∣a−cd∣<12d2时，cd就是a的连分数收敛当满足|a - \frac{c}{d}| < \frac{1}{2d^2} 时，\\ \frac{c}{d} 就是 a 的连分数收敛 当满足∣a−dc​∣<2d21​时，dc​就是a的连分数收敛

在RSA上的应用

对于公钥e和私钥d:

由e⋅d≡1 (mod LCM(p−1,q−1))知∃K,st.e⋅d=1+K⋅LCM(p−1,q−1))令G=GCD(p−1,q−1)有e⋅d=1+KG(p−1)(q−1)令k=KGCD(K,G),g=GGCD(K,G)有e⋅d=1+kg(p−1)(q−1)此时，epq=kdg(1−δ),其中δ=p+q−1−gkpq这时只要δ足够小，epq就可以近似看成kdg，而epq是我们已知的由e \cdot d \equiv 1 \ (mod \ LCM(p-1, q-1))\\ 知\exists K, st. e \cdot d = 1 + K \cdot LCM(p-1, q-1))\\ 令G = GCD(p-1, q-1)\\ 有e \cdot d = 1 + \frac{K}{G} (p-1)(q-1)\\ 令k = \frac{K}{GCD(K,G)}, g = \frac{G}{GCD(K,G)}\\ 有e \cdot d = 1 + \frac{k}{g} (p-1)(q-1)\\ 此时，\frac{e}{pq} = \frac{k}{dg}(1-\delta),其中\delta = \frac{p+q-1-\frac{g}{k}}{pq}\\ 这时只要\delta 足够小，\frac{e}{pq} 就可以近似看成 \frac{k}{dg}，而\frac{e}{pq}是我们已知的\\ 由e⋅d≡1 (mod LCM(p−1,q−1))知∃K,st.e⋅d=1+K⋅LCM(p−1,q−1))令G=GCD(p−1,q−1)有e⋅d=1+GK​(p−1)(q−1)令k=GCD(K,G)K​,g=GCD(K,G)G​有e⋅d=1+gk​(p−1)(q−1)此时，pqe​=dgk​(1−δ),其中δ=pqp+q−1−kg​​这时只要δ足够小，pqe​就可以近似看成dgk​，而pqe​是我们已知的
